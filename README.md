# Readbookapp

App về đọc sách online.

Chức năng: đăng nhập, đăng kí, tìm kiếm, xem thông tin cá nhân, đọc sách, chọn chương, chuyển tiếp chương, sửa thông tin cá nhân, trang thể loại.
